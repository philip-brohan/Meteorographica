Authors and acknowledgements¶

This package is currently maintained by Philip Brohan (philip.brohan @ metoffice.gov.uk). All criticism should be directed to him - put please don’t send email, raise an issue instead.


All criticism should go to the maintainer; credit is more widely distributed:
