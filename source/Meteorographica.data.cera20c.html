Meteorographica.data.cera20c.fetch module¶

The functions in this module fetch CERA 20C data from ECMWF and store it on $SCRATCH.

Uses the ECMWF Public data API:
https://software.ecmwf.int/wiki/display/WEBAPI/ECMWF+Web+API+Home
