Meteorographica: python code for plotting weathermaps¶

This code library extends SciTools (Iris and Cartopy) in providing tools for plotting meteorological data. In particular it supports plotting synoptic charts - maps of instantanious surface weather, and indicating the uncertainties in those charts.
